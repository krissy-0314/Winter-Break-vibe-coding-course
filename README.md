## Week 1｜單元一：Vibe Coding 入門

### 📋 課程主題
**打造你的第一個 AI 協作專案** — 拆解問題 + 環境建置 + 驗收思維

### 🎯 學習重點
- 學會將模糊需求拆解為 IPO（輸入 → 處理 → 輸出）
- Vibe Coding 核心：不是拼命寫 Code，而是精準提問與驗收
- 黃金法則 3×3：3 個成功案例 + 3 個失敗案例
- AI 時代開發迴圈：Prompt → Generate → Verify → Iterate

## 🔑 核心概念

### Vibe Coding 開發流程
```
Spec (I/O) → AI (TDD) → Validate → Git → Deploy
```

### IPO 思維框架
- **I (Input)**: 輸入什麼？
- **P (Process)**: 處理邏輯？
- **O (Output)**: 輸出什麼？
- **AC**: 驗收標準 3×3
